# Modific

Modific is a Light Table plugin for highlighting lines changed since the last commit.

# Features

☑ Show gutter marks  
☑ Jump between changes (`Ctrl+Shift+PageUp/PageDown`)  
☑ Show original part (`Ctrl+Alt+c`)
